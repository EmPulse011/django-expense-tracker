# Django Expense Tracker

A web-based expense tracker built with Django and Bootstrap 5. Users can log expenses, set monthly budgets, and filter spending by month.

## Features

- Add, edit, and delete expenses
- Set and update monthly budgets
- Filter expenses by month
- Responsive UI built with Bootstrap 5
- Budget vs. spending overview

## Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, Bootstrap 5
- **Database:** SQLite (default Django dev database)
